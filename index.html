☀️ 기상청 날씨 달력 자동화 제작 가이드

1 프로젝트 내 계정으로 가져오기 (Fork)

이 저장소 오른쪽 상단의 [Fork] 버튼을 눌러 본인의 GitHub 계정으로 복사하세요.

2 기상청 API 인증키 발급

1. 기상청 API 허브에 접속하여 회원가입을 진행합니다.

2. 로그인 후 상단 메뉴의 [마이페이지] → [인증키 현황]에서 나만의 인증키를 복사해두세요.

3 GitHub에 인증키 저장하기

1. 내 GitHub 저장소의 [Settings] → [Secrets and variables] → [Actions]로 이동합니다.

2. [New repository secret] 버튼을 클릭합니다.

3. Name에 KMA_API_KEY를 입력하고, Value에 방금 복사한 인증키를 붙여넣어 저장합니다.

4 기상청 API 활용신청 (중요)

왼쪽 메뉴에서 다음 항목들을 찾아 각각 [API활용신청] 버튼을 누르세요.

🏠 단기예보: 예특보 → 단기예보 → 4.3 단기예보조회
🌡️ 중기예보 (2곳 필수): 예특보 → 중기예보 → 2.2 중기기온조회 예특보 → 중기예보 → 2.3 중기육상예보조회

* 신청 즉시 승인되며, 마이페이지에서 상태가 '사용'인지 확인하세요.

5 내 동네 설정값 찾기

동네 이름을 입력하면 코드에 넣을 좌표값을 계산해드립니다.

👇 6단계에서 사용할 값들입니다:

  • NX: 60
  • NY: 127
  • 기온구역(REG_ID_TEMP): 11B10101
  • 육상구역(REG_ID_LAND): 11B00000

6 내 동네 정보 입력하기

1. 내 저장소에서 update_calendar.py 파일을 클릭합니다.

2. 오른쪽 상단의 연필 아이콘(Edit)을 눌러 수정 모드로 들어갑니다.

3. 상단 설정 부분에 5단계에서 얻은 좌표와 구역코드를 그대로 입력한 후 하단의 [Commit changes...]를 눌러 저장합니다.

7 캘린더 주소 복사 및 구독

1. 내 저장소 메인에서 weather.ics 파일을 클릭합니다.

2. 오른쪽 상단의 [Raw] 버튼을 누르면 나오는 텍스트 화면의 브라우저 주소를 복사합니다.

3. 아이폰이나 갤럭시 달력의 '구독 캘린더' 메뉴에 해당 주소를 붙여넣으면 끝!
